# jms-cli

> js-multi-seed 标准开发/管理工具

### 开始使用

安装:

```bash
npm install -g jms-cli
```

创建一个新项目:

```bash
jms create my-project 或 jms c my-project
```

添加一个已存在的项目:

```bash
jms add D:\my-project 或 jms a D:\my-project
```

项目管理控制面板:

```bash
jms console 或 jms cb
```

列出本机所有的项目:

```bash
jms list 或 jms ls
```

移除项目（仅移除项目记录，不删硬盘上的文件）:

```bash
jms remove 或 jms rm
```

更新项目基本信息:

```bash
jms refresh 或 jms re
```

更换 npm 源:

```bash
jms source 或 jms s
```
